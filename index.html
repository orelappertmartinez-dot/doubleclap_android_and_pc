🚀 Instalación y Configuración

Para Windows (Ordenador):

  1. Descarga los archivos del código Python
  2. Instala las dependencias:
    pip install numpy sounddevice
  3. Ejecuta con configuración:
    python clap-trigger.py --configure
  4. Selecciona las aplicaciones que deseas abrir con palmadas
  5. Presiona "Iniciar" y comienza a usar

Para Android:

  1. Compilar la APK:
    • Instala Docker Desktop
    • Haz clic en el botón "🐳 Compilar con Docker"
    • Lee el archivo DOCKER_BUILD.md para instrucciones detalladas
    • En Windows, ejecuta build-apk.bat
    • En Linux/Mac, ejecuta ./build-apk.sh
    • El APK se generará en la carpeta bin/ (demora ~15-20 minutos)
  2. Transfiere el APK a tu teléfono Android
  3. Abre el archivo y instala la aplicación
  4. Otorga permisos de micrófono
  5. Abre la aplicación y configura las apps deseadas
  6. Presiona "Iniciar Clap Trigger" para comenzar

🐳 Compilar con Docker (RECOMENDADO - Más Fácil):

  1. Instala Docker Desktop
  2. En Windows, ejecuta:
    build-apk.bat
  3. En Linux/Mac, ejecuta:
    chmod +x build-apk.sh && ./build-apk.sh
  4. Espera 15-20 minutos (depende de tu conexión y PC)
  5. El APK se generará en: bin/claptrigger-0.1-debug.apk

Compilar Manualmente (Usuarios Avanzados):

  1. Instala Buildozer en Linux:
    pip install buildozer Cython
  2. Instala dependencias del sistema:
    sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev
  3. Ejecuta buildozer:
    buildozer android debug
  4. El APK se generará en: bin/

❓ Preguntas Frecuentes

¿Qué hago si no detecta palmadas?

Intenta ajustar el umbral de sensibilidad. Un valor más bajo hace la detección más sensible. Prueba valores entre 20 y 40.

¿Necesito conexión a internet?

No, la aplicación funciona completamente offline. Solo necesita acceso al micrófono de tu dispositivo.

¿Qué permisos necesita?

Solo necesita permiso de acceso al micrófono para detectar palmadas.

¿Puedo personalizar qué aplicaciones se abren?

Sí, puedes agregar, eliminar o cambiar cualquier aplicación desde la interfaz de configuración.

¿Es compatible con mi dispositivo?

Android: Android 7.0 o superior
Ordenador: Windows 7+, Linux, o macOS con Python 3.7+
